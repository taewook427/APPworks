UnitConv : 단위 환산기로 위장한 비밀번호 관리앱.

앱 시작 시 미국 단위계과 미터법 환산 화면이 나옵니다.
버튼을 두번 눌러 역변환을 수행합니다.
길이 : in, ft, yd, mi, cm, m, km
무게 : oz, lb, g, kg
기타 : 'F, 'C, K, gal, L

입력 숫자로 미리 설정된 워드를 사용하면 비밀번호 관리 화면이 나옵니다.
최초 비밀번호는 0000 이며, 000000000000 으로 저장 데이터를 삭제할 수 있습니다.

관리 화면에서는 여러 페이지로 구성된 노트를 보거나 수정할 수 있습니다.
각 페이지는 제목과 본문으로 이루어지며, 주제별로 다른 페이지를 사용하면 됩니다.
